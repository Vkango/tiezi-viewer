# tiezi-viewer

将保存的JSON数据转换为HTML输出，基于Python + HTML + CSS。

如果需要下载器请移步[这里](https://github.com/Vkango/tiezi-transfer/blob/master/README.md)

# 🚀 清单

## 回复类型支持

* [x] 文本
* [x] 图片
* [x] 链接
* [x] at人
* [ ] 视频
* [ ] 语音

## 功能支持

* [ ] 优先本地数据源，若不存在则使用在线资源
* [ ] 上一页、下一页、跳页 (JS实现)
* [ ] 黑暗模式 (这个可以先用DarkReader等插件)
